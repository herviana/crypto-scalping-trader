# ⚡ Crypto Scalping Trader — Bankr Skill

Skill untuk **Bankr Agent** yang mengotomasi trading crypto dengan metode **scalping** di semua chain.

## ✨ Fitur

- ⚡ Trading otomatis dengan metode scalping
- 🔍 Scan semua chain untuk peluang scalping terbaik
- 📊 Analisis indikator otomatis (RSI, EMA, Volume, MACD)
- 🛡️ Manajemen risiko otomatis (stop loss, max loss harian)
- 📈 Laporan profit/loss harian
- 🔗 Terintegrasi dengan skill x-trending-crypto dan token-scam-analysis

## 📦 Instalasi

### Di Bankr (HP/Web)
1. Buka Bankr → **Agent Settings** → **Skills**
2. Tap **"+ Add"**
3. Paste URL:
```
https://github.com/herviana/crypto-scalping-trader
```
4. Konfirmasi install ✅

## 🚀 Cara Pakai

```
Cari koin bagus untuk scalping hari ini
Auto scalping ETH dengan modal $50
Scan semua chain untuk peluang scalping
Cek posisi scalping saya yang terbuka
Tampilkan laporan scalping hari ini
Exit semua posisi scalping sekarang
```

---

## ⚠️ Manajemen Risiko

- Max per trade: 10% portfolio
- Max loss harian: 5% portfolio
- Stop loss: WAJIB di setiap trade
- Max posisi terbuka: 3 bersamaan
